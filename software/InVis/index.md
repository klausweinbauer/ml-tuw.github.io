## Using it
 
### Download
    wget {{site.url}}{{page.url}}InVis_v1.tar.bz2
    tar xfj InVis_v1.tar.bz2
    
### Install
    

#### With Conda
    cd InVis
    conda create --name invis --file spec-file.txt 

### Run
    conda activate invis
    python Main.py
